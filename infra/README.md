# Terraform Infrastructure POC

## Setup
1. Install Terraform.
2. Configure AWS Credentials (`aws configure`).
3. Navigate to `infra/envs/dev`.
4. Run `terraform init`.
5. Run `terraform plan`.
6. Run `terraform apply`.

## Modules
- VPC
- EC2
- RDS
- Security Groups

## Environment
- Dev: `infra/envs/dev`
